# Bookstore

Proyecto de e-commerce de libros para el curso de <b>React</b> de CODERHOUSE.

<li>Realizado con create-react-app. </li>

<li>Se utilizó la librería <a href="https://react-icons.github.io/react-icons/"><b>react-icons</b></a> para los íconos y <a href="https://styled-components.com"><b>styled components</b></a> para los estilos, ya que facilita la reutilización y permite escribir estilos en el jsx, ahorrandose unos cuantos archivos extra.</li>

<li>Las ilustraciones utilizadas fueron sacadas de <a href="http://www.canva.com">Canva</a></li>

<li>Se utilizó <a href="">react-hook-form</a> para validar los formularios</li>
